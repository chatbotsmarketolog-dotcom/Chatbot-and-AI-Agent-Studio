Как получить токен:
1. Перейди: github.com/settings/tokens
2. Нажми Generate new token (classic)
3. Введи название: КРИСТИНА.AI CMS
4. Поставь галочку: repo (полный доступ)
5. Нажми Generate token
6. Скопируй токен (начинается на ghp_...) и вставь ниже
